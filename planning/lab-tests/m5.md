# Lab-test evidence — M5: Docs Assistant (Naive RAG)

**Machine:** Apple Silicon (arm64), 16 GB · Rancher Desktop · **native Ollama**
**Date:** 2026-07-05
**Stack:** ChromaDB `0.5.20` (container) + Streamlit app (container) + native Ollama
**Models:** `qwen2.5:1.5b` (generation) + `nomic-embed-text` (embeddings, 768-dim) — both native.

> Apple-Silicon pattern: model server NATIVE; ChromaDB + app containerized, reaching Ollama at
> `host.docker.internal:11434`. This is the reuse `lightweight-genai-stack` adapted off its
> *containerized* Ollama onto native Ollama, and pinned Chroma to 0.5.x to match the client.

## ChromaDB container — up

```

$ curl -s localhost:8000/api/v2/heartbeat   ->  {"nanosecond heartbeat": 1783255100...}
```

## Native embeddings — work

```
$ curl -s localhost:11434/api/embeddings -d '{"model":"nomic-embed-text","prompt":"..."}'
   ->  embedding with 768 dimensions
```

## Full naive-RAG round-trip — validated headlessly

Ingested 3 Acme runbook chunks (embedded via `nomic-embed-text`) into a Chroma collection, then
queried and generated a grounded answer with `qwen2.5:1.5b`:

```
[ingest]   stored 3 chunks in Chroma collection 1c610dd7
[retrieve] top chunk (dist=164.777):
           "To restart the Acme payments service, run: kubectl rollout restart deploy/payments -n prod."
[generate] "To restart the payments service, you need to execute the following command:
            kubectl rollout restart deploy/payments -n prod"
```

The query *"How do I restart the payments service?"* retrieved the correct runbook chunk by semantic
similarity and the model answered **grounded in that retrieved context** — the whole naive-RAG loop
(ingest → embed → store → retrieve → generate) working on native Ollama + a containerized vector DB.

## App image — trimmed for the laptop budget

`main.py` imports only the `langchain-ollama` / `langchain-chroma` stack — **not**
`sentence-transformers`/`torch`. Dropped `sentence-transformers` (≈2 GB torch) and unused
`python-docx` from `requirements.txt`, keeping the RAG stack + `pypdf` + `unstructured`/`markdown`
loaders. (App build + Streamlit health validation appended below.)

App build + full-stack health (trimmed image):


✅ The Docs Assistant image builds and runs on Rancher Desktop, serving the Streamlit UI on :8501,
wired to native Ollama (`host.docker.internal:11434`) + ChromaDB. `docker compose down` frees RAM.
